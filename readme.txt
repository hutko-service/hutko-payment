Hutko payment gateway for WooCommerce.

== Description ==

Hutko Payments plugin allows WooCommerce stores to accept payments via Hutko service.

== Installation ==

1. Upload the `hutko-payments` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure settings under WooCommerce > Settings > Payments.
